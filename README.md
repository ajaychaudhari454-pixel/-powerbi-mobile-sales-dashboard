# Power BI Mobile Sales Dashboard

## Project Overview
This Power BI dashboard provides an analysis of mobile phone sales across multiple countries.

## Countries Covered
- India
- USA
- UAE
- UK

## Key Metrics
- Total Sales
- Total Profit
- Sold Units
- Monthly Sales Trend
- Customer Count

## Dashboard Features
- Country-wise sales analysis
- Monthly sales trend visualization
- Brand and selling price slicers
- City-wise contribution
- Interactive visuals and filters

## Tools Used
- Power BI Desktop
- Excel (Data Source)
